観光地

TOUR

雄大な自然と伝統文化が息づく魅力あふれる場所を紹介します。
鳴門の渦潮や祖谷渓、阿波踊りなど、ここでしか体験できない感動をお楽しみください。

●鳴門公園

壮大な渦潮が間近で見られるスポットとして知られ、その迫力ある光景に圧倒されます。展望台や遊歩道からは、瀬戸内海と太平洋が交わる絶景を眺めることができ、自然美を存分に満喫できます。また、夕暮れ時には美しいサンセットが広がり、ロマンチックな雰囲気が漂うため、訪れるたびに異なる魅力が感じられる場所です。

営業時間 9:00 - 18:00
住 所 〒772-0053
徳島県鳴門市鳴門町土佐泊浦福池

●大塚国際美術館

世界の名画を陶板で再現した、日本で唯一の「陶板名画美術館」です。広大な館内には、古代から現代までの美術史をたどる1000点以上の作品が展示され、世界旅行をするような感覚で楽しめます。また、作品に直接触れることができるため、美術に親しみやすく、家族や友人と一緒に訪れても学びと感動が得られる魅力的なスポットです。

営業時間 9:00 - 17:00
